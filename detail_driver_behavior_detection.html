Ứng dụng học sâu sử dụng các mô hình CNN để phân tích và phân loại hành vi của tài xế thông qua ảnh hoặc video. Mục tiêu là hỗ trợ tăng cường an toàn giao thông bằng cách phát hiện sớm các hành vi nguy hiểm như nhắn tin, nói chuyện điện thoại khi đang lái xe.

A deep learning application using CNN models to analyze and classify driver behaviors from images or videos. The goal is to enhance traffic safety by detecting risky actions such as texting or talking on the phone while driving.

Các Lớp Hành Vi Behavior Classes

Những hành vi được phân loại Classified Behaviors

01

Other

Các hành vi khác không thuộc nhóm nguy hiểm hoặc an toàn. Other behaviors that are not explicitly categorized as safe or risky.

02

Safe

Lái xe tập trung, hai tay trên vô lăng, không sử dụng điện thoại. Driver is focused, hands on the wheel, not using a phone.

03

Talking

Tài xế đang nói chuyện điện thoại khi lái xe. Driver is talking on the phone while driving.

04

Texting

Tài xế đang nhắn tin trên điện thoại, mất tập trung khỏi đường đi. Driver is texting on the phone and losing focus from the road.

05

Turning

Tài xế đang rẽ hoặc quay đầu, thay đổi hướng lái. Driver is turning or changing direction.

Pipeline

Cách hệ thống hoạt động How the system works

1

Tải mô hình an toàn Safe model loading

Sử dụng st.cache_resource để tải và cache các mô hình, tự động tải file từ Google Drive nếu chưa tồn tại. Uses st.cache_resource to load and cache models, automatically downloading from Google Drive if missing.

2

Tiền xử lý ảnh Image preprocessing

Chuẩn hóa ảnh về kích thước 224x224, chuyển sang RGB, scale về khoảng \[0, 1\] và thêm batch dimension. Normalizes image to 224x224, converts to RGB, scales to \[0, 1\] and adds batch dimension.

3

Suy luận với mô hình Model inference

Mô hình CNN nhận ảnh đã tiền xử lý và trả về vector xác suất cho từng lớp hành vi. CNN model receives the preprocessed image and outputs probability scores for each behavior class.

4

Hiển thị kết quả Display results

Streamlit hiển thị nhãn dự đoán, biểu đồ xác suất và (với video) slider để xem từng khung hình. Streamlit shows predicted label, probability chart and (for videos) a slider to inspect each frame.

app.py 
import streamlit as st
from tensorflow.keras.models import load_model

@st.cache_resource
def load_models_safely():
    models = {}
    # Khai báo thông tin model và ID Google Drive
    for name, info in model_info.items():
        # Tải model nếu chưa tồn tại
        if not os.path.exists(info['path']):
            st.info(f"Downloading {name} model...")
            gdown.download(id=info['id'], output=info['path'], quiet=False)

        try:
            models[name] = load_model(info['path'])
        except Exception as e:
            st.error(f"Không thể load model {name}: {e}")

    return models

Mô hình CNN CNN Models

So sánh các mô hình Model comparison

AlexNet

Nhẹ & Nhanh Lightweight & Fast

Kiến trúc 8 tầng (5 convolution + 3 fully-connected), phù hợp cho môi trường tài nguyên thấp. 8-layer architecture (5 conv + 3 dense), suitable for low-resource environments.

  • Tốc độ suy luận cao High inference speed
  • Độ chính xác vừa phải Moderate accuracy

InceptionV3

Độ chính xác cao High Accuracy

Sử dụng các Inception module để xử lý nhiều kích thước filter song song, cân bằng hiệu quả tính toán. Uses Inception modules with parallel filters of different sizes, balancing computation efficiency.

  • Độ chính xác cao hơn AlexNet Higher accuracy than AlexNet
  • Nặng hơn về tài nguyên Heavier in resources

GoogLeNet (Inception V1)

Cân bằng tối ưu Best Trade-off

Đạt được cân bằng rất tốt giữa tốc độ, kích thước model và độ chính xác, phù hợp cho ứng dụng thực tế. Provides an excellent balance between speed, model size and accuracy for real-world applications.

Thường là lựa chọn tối ưu trong dự án này. Often the best choice in this project.

Troubleshooting

Lỗi thường gặp Common issues

  • ModuleNotFoundError

    Đảm bảo đã cài đặt đầy đủ package với: Make sure all packages are installed using: 

    pip install -r requirements.txt

  • Lỗi tải mô hình / Model download error

    Kiểm tra kết nối Internet và đảm bảo file trên Google Drive được chia sẻ công khai với ID chính xác. Check your Internet connection and ensure Google Drive files are public and IDs are correct.

  • Out of memory

    Khởi động lại ứng dụng, giảm kích thước batch hoặc chạy trên môi trường có RAM cao hơn (>=2GB). Restart the app, reduce batch size or use an environment with higher RAM (>=2GB).
