Dữ liệu Dataset

Thông tin bộ dữ liệu Dataset overview

Dữ liệu gồm 10.000 khách hàng trong file Churn_Modelling.csv với các thuộc tính nhân khẩu học, tài chính và hành vi sử dụng dịch vụ, cùng nhãn mục tiêu Exited (1: rời bỏ, 0: ở lại). The dataset contains 10,000 customers in Churn_Modelling.csv, including demographic, financial, and behavioral features plus a target label Exited (1: churn, 0: stay).

  • CreditScore, Age, Tenure, Balance
  • NumOfProducts, HasCrCard, IsActiveMember, EstimatedSalary
  • Geography (France, Spain, Germany), Gender (Male/Female)
  • Exited – churn label
Tính năng Features

Những điểm chính của dự án Project highlights

Xử lý & mã hóa dữ liệu Data preprocessing & encoding

Sử dụng One-Hot Encoding cho Geography, Gender, chọn tập biến đầu vào có ý nghĩa cho mô hình. Uses One‑Hot Encoding for Geography and Gender, selecting the most informative input features.

Cân bằng dữ liệu với SMOTE Handling imbalance with SMOTE

Áp dụng SMOTE trên tập train để tăng số lượng mẫu lớp churn, giúp mô hình học tốt hơn nhóm thiểu số. Applies SMOTE on the training set to oversample churn cases, improving performance on the minority class.

Huấn luyện nhiều mô hình khác nhau Training multiple models

So sánh Decision Tree, Random Forest, XGBoost và mô hình tổ hợp VotingClassifier. Compares Decision Tree, Random Forest, XGBoost and an ensemble VotingClassifier.

Đánh giá bằng nhiều chỉ số quan trọng Rich evaluation metrics

Sử dụng Accuracy, F1, Recall, Precision cho cả hai lớp, tập trung tối ưu Recall/F1 của lớp churn. Uses Accuracy, F1, Recall and Precision for both classes, focusing on improving churn class Recall/F1.

Pipeline

Quy trình xây dựng mô hình Modeling pipeline

1

Khám phá & xử lý dữ liệu EDA & preprocessing

Đọc dữ liệu, kiểm tra thiếu, phân phối biến, tương quan và chuẩn hóa các trường đầu vào. Loads data, checks missing values, explores distributions, correlations and prepares input features.

2

Chia train/test & SMOTE Train/test split & SMOTE

Dùng train_test_split (test 20%, stratify), sau đó áp dụng SMOTE trên tập train. Uses train_test_split (20% test, stratified), then applies SMOTE on the training set.

3

Huấn luyện mô hình & Voting Train models & Voting

Train Decision Tree, Random Forest, XGBoost rồi kết hợp bằng VotingClassifier. Trains Decision Tree, Random Forest, XGBoost and combines them using a VotingClassifier.

4

Đánh giá & so sánh Evaluate & compare

Dùng hàm evaluate_model để tính Train/Test Score, F1, Recall, Precision và tổng hợp kết quả trong results_df. Uses evaluate_model to compute Train/Test Score, F1, Recall, Precision and aggregate results into results_df.

customer_churn.ipynb 
from sklearn.ensemble import RandomForestClassifier, VotingClassifier
from sklearn.tree import DecisionTreeClassifier
from xgboost import XGBClassifier

dt_model = DecisionTreeClassifier(ccp_alpha=0.000949, random_state=0)
rf_model = RandomForestClassifier(
    n_estimators=51,
    max_depth=11,
    min_samples_split=4,
    min_samples_leaf=1,
    random_state=0,
)
xgb_model = XGBClassifier(
    min_child_weight=0.8,
    reg_alpha=0.4,
    reg_lambda=1.1,
    random_state=0,
)

voting_clf = VotingClassifier(
    estimators=[
        ("dt", dt_model),
        ("rf", rf_model),
        ("xgb", xgb_model),
    ],
    voting="hard",
)

VotingClassifier cho Test Score và F1 Score cao nhất, đồng thời giữ Train Score ở mức vừa phải, giảm nguy cơ overfit so với XGBoost đơn lẻ. The VotingClassifier achieves the best Test Score and F1 Score while keeping Train Score moderate, reducing overfitting compared to standalone XGBoost.
